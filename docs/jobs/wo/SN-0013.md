## Objective

Give each tenant a nullable default warehouse and a stock-by-warehouse view.
Onboarding sets the default to the sole created warehouse (ADR-0011); owners can
set/clear it from Settings and from the catalog warehouse list (Default badge).
The receive, delivery, and opname forms prefill the warehouse from it, still
overridable per transaction. Add a stock-by-warehouse view (on-hand qty, average
cost, value per product per warehouse, filterable by warehouse, with a total)
backed by the existing `/reports/stock-valuation` endpoint.
