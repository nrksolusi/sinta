## Objective

Let related documents be inspected inline instead of via navigation, and move
dense secondary panels behind disclosure primitives.

## Required reading (weighted)

- **Must:** `../../design/ui-design-principles.md` principle 9;
  `../../plans/fix-4-ui-ux-revamp.md` Slice 7 + Ground-truth ("no `ui/accordion.tsx` /
  `ui/collapsible.tsx`", `DataTable` already supports `renderExpandedRow` /
  `expandedRowId` at `data-table.tsx:31-48,93-113`).

## Scope fence

**In scope** - generate `ui/accordion.tsx` + `ui/collapsible.tsx`; linked-document
peek showing line-detail-only inline (PO -> receipts, SO -> deliveries, opname ->
kartu stok) via accordion / DataTable expandable row, with a link to the full
document; sweep touched editors/detail pages to move dense/secondary panels into
collapsibles.

**Out of scope** - server changes; redesigning the full linked document (peek is
line-detail-only); the wizard/dashboard WOs.
